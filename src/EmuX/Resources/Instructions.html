  • AAA

    If the low nibble of the AL register is less than 9 or AF is true then add 6 to AL, increment AH and set AF and CF to true

    If not set AF and CF to false


  • AAD

    Add the AH and AL register and save the result to the AL register


  • AAM

    Set the value of the AH register equal to the result of AL divided by 10 and save the remainder to the AL register


  • AAS

    If the low nibble of the AL register is bigger than 9 or AF is true then subtract 6 from AL, decrement AH and set AF and CF to true

    If not set AF and CF to false


  • ADC

    Save to the destination the value of destination + source + CF


  • ADD

    Save to the destination the value of destination + source


  • AND

    Save to the destination the value of the AND bitwise operation of destination AND source


  • CALL

  • CBW

    If the first bit of AL is 1 then set AH to 255

    Else set AH to 0


  • CLC

    Set the CF to false


  • CLD

    Set the DF to false


  • CLI

    Set the IF to false


  • CMC

    Execute the NOT operation to the CF


  • CMP

    Compare the destination value with the source value and set the appropriate flags depending on the comparison made


  • CWD

    If the first bit of AX is 1 then set DX to 65535

    Else set DX to 0


  • DAA

    If lower nibble of AL is bigger than 9 or AF is true then add 6 to AL and set AF to true

    Else if AL is bigger than 159 or CF is true then add 96 to AL and set CF to true


  • DAS

    If lower nibble of AL is bigger than 9 or AF is true then subtract 6 from AL and set AF to false

    Else if AL is bigger than 159 or CF is true then subtract 96 from AL and set CF to false


  • DEC

    Decrement one from the value provided and save the result to the destination


  • DIV

    If the operand given is a byte then set AL to AX divided by the operand and set AH to the remainder

    Else set AX to DX:AX divided by the operand and set DX to the remainder

  • LAHF

    Set the value of AH equal to the flags register value

    8th bit = high most bit

    8th bit = SF

    7th bit = ZF

    6th bit = 0

    5th bit = AF

    4th bit = 0

    3th bit = PF

    2th bit = 1

    1th bit = CF
